# Changelog

All notable changes to the Digital Signal Governance (DSG) public conceptual framework repository are documented here.

Versioning follows Semantic Versioning principles adapted for conceptual framework documentation:

- **Major versions** — foundational category redefinition, structural taxonomy restructuring, or boundary model changes
- **Minor versions** — addition of new terminology, taxonomy branches, boundary documentation, or conceptual framework extensions
- **Patch versions** — editorial clarifications, metadata corrections, citation updates, documentation refinements, and non-substantive maintenance changes
